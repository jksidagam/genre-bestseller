=== Genre Bestseller Plugin ===

Description:

This is a wordpress plugin to create a Gutenberg block that generates a best seller book from a genre belonging to a selection of genres from API.


API documentation source: https://developer.penguinrandomhouse.com/docs

---------------------

General Instructions:

Before activating the plugin, from the plugin folder, run the below command to install node modules

npm install

After the installation of the node modules, run the below command to generate the build

npm start

----------------------
